Manual de Instruções

Ao executar o código, será lido o arquivo de entrada, montado a grade de horário e exibido na tela a grade montada.
Em seguida o código pede para apertar Enter para dar continuidade.

Assim, será exibido na tela um menu com as seguintes opções:

1 - Conferir horários por período
Solicita ao usuário o período desejado e em seguida exibe a grade montada para aquele período.

2 - Conferir horários por professor
Solicita ao usuário o nome do professor e em seguida exibe a grade montada para aquele professor. Caso o professor pesquisado não exista na base retorna erro.

9 - Sair
Exibe uma mensagem de agradecimento e finaliza o código.

Demais opções não existentes retornam erro.

Nota: Para execução do código, faz-se necessário ter um arquivo "entrada.txt" no diretório do projeto. Caso o arquivo não esteja lá o código executará normalmente mas não terá base de dados para montar a grade de horários. O arquivo deve seguir o seguinte modelo:

nome_da_disciplina nome_do_professor período
